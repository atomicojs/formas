# Welcome to Formas
### a system of components to exemplify the use of Atomico + Storybook

Formas exemplifies the workflow of Atomico + Storybook, some objectives of this library are:

1. Monorepository of components
2. Storybook 7
3. @atomico/storybook: Improves the experience of using Atomico in Storybook.
4. Automatic publishing of components in NPM.
5. Automatic wrapper for React, Preact and Vue of the exported webcomponents
