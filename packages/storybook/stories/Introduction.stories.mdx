# Bienvenido a Formas 
### un sistema de componentes para ejemplificar el uso de Atomico + Storybook

Formas busca ejemplificar el flujo de trabajo de Atomico + Storybook, buscando resolver los siguientes objetivos:

1. Monoreposorio: La expriencia de desarrollo  fluida  gracias a  @atomico/run, un CLI que permite ejecutar script de NPM de una forma eficiente 
2. Storybook 7: Utiliza la ultima version de Storybook
3. @atomico/storybook: Mejora la experiencia de uso de Atomico en Storybook.
4. Publicacion automatica de componentes.
5. Wrapper para React, Preact y Vue de los webcomponents exportados
